# Laboratorio Módulo Layout - EXTRA

##

**npm run dev:extra** para el laboratorio-extra

### EJERCICIO único

### INTRODUCCIÓN

Vamos a crear un Layout para una aplicación, que contendrá:

- Header:
  - Toolbar con un input de texto para búsquedas y nombre del usuario logado.- Barra con el nombre de la aplicación.
  - Nav:- Menú con varias opciones de navegación- Main:
  - Contenido con la información principal de la aplicación.
  - Fondo diferente de blanco.- Footer:- Texto de la compañía, al final de la página (aunque el contenido no ocupe todo el alto)

### ACCIONES DE USUARIO

Hacer que el navegador sea más grande o pequeño para ver como cambian los elemenots.
